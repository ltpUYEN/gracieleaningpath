NETFLIX Project Analysis

This project involves an in-depth analysis of the Netflix dataset, exploring trends in content production, genre popularity, and rating distributions across different countries and years.

Data Source & Tech Stack:
Kaggle: Netflix Movies and TV Shows dataset.
Tableau: Used for data cleaning, analysis, and creating the visualizations.

Key Findings & Insights:

  • The Netflix library analyzed consists significantly more of movies (~70%) than TV shows (~30%).
  • Content availability is heavily concentrated in certain countries, particularly the United States.
  • There was a marked increase in the number of titles added annually in the late 2010s (peaking around 2018-2020).
  • Popular genres include combinations involving Dramas, International Movies, Comedies, and Documentaries.

Project Disclaimer:

This dashboard was created as a study project using a Netflix dataset from Kaggle, primarily for the purpose of learning Tableau.
